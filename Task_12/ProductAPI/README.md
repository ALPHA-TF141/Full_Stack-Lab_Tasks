# 🛍️ Task 12: Product Management REST API

---

## 📋 Description
This project is a **Spring Boot REST API** for managing products.

Supports:
- GET (fetch products)
- POST (add product)
- PUT (update product)
- DELETE (remove product)

All responses are in **JSON format**.

---

## 🚀 How to Run

### 1. Start Application
```bash
mvn spring-boot:run
